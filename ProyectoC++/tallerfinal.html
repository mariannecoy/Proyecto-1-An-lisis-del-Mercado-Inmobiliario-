Proyecto Final Integrado: Sistema de Predicción y Gestión de Productos con C++ y Java

Objetivo:

El objetivo del proyecto es que los estudiantes desarrollen una aplicación integrada que combine la implementación de regresión lineal en C++ para predecir precios con un sistema de gestión de productos en Java, aplicando los conceptos de Programación Orientada a Objetos (POO). El producto final puede ser presentado con una interfaz gráfica o con una interacción a través de la consola.

Estructura del Proyecto

El proyecto debe seguir la siguiente estructura de carpetas y archivos, que debe ser versionado en GitHub:

ProyectoFinal/
├── C++/
│   ├── main.cpp                    // Implementación principal en C++
│   ├── regresion_lineal.h          // Archivo de cabecera para la regresión lineal
│   └── regresion_lineal.cpp        // Implementación de la regresión lineal
├── Java/
│   ├── src/
│   │   ├── controllers/
│   │   │   └── VentasController.java  // Controlador de la interfaz gráfica o consola
│   │   ├── models/
│   │   │   ├── Producto.java           // Clase Producto (base)
│   │   │   ├── ProductoElectronico.java // Clase derivada ProductoElectrónico
│   │   │   ├── ProductoAlimenticio.java // Clase derivada ProductoAlimenticio
│   │   │   └── PrediccionPrecios.java   // Conexión con la regresión lineal en C++
│   │   └── views/
│   │       └── VentasView.fxml          // Archivo FXML para la interfaz gráfica (solo si aplicable)
│   └── Main.java                     // Punto de entrada principal para el sistema en Java
├── README.md                         // Instrucciones y detalles del proyecto
├── .gitignore                        // Archivos que no deben incluirse en el repositorio
└── mse.txt                           // Archivo donde se registran los errores cuadráticos medios (si es necesario)

Alcance del Proyecto:

  1. Parte 1 (C++): Predicción de Precios con Regresión Lineal

    • Implementar la regresión lineal utilizando C++ para predecir el precio de productos en función de ventas previas.
    • Integrar el cálculo de la regresión lineal con el sistema en Java.
    • Los resultados deben ser mostrados en la interfaz gráfica (opción 1) o en la consola (opción 2).

  2. Parte 2 (Java): Sistema de Gestión de Productos

    • Desarrollar un sistema de gestión de productos utilizando Java y Programación Orientada a Objetos.
    • El sistema debe permitir al usuario:
      • Registrar productos con atributos como nombre, precio base y cantidad disponible.
      • Registrar ventas de los productos.
      • Mostrar la lista de productos y sus precios.
      • Calcular el precio de productos a través de la predicción de precios, integrando la regresión lineal en C++.
    • Este sistema puede ser presentado con una interfaz gráfica (opción 1) o con una interacción en consola (opción 2).

Opciones de Presentación:

Opción 1: Presentación con Interfaz Gráfica (Base 5)

  • Los estudiantes deben implementar una interfaz gráfica para la gestión de productos y la predicción de precios utilizando JavaFX.
  • La interfaz gráfica debe permitir al usuario realizar las siguientes acciones:
    • Agregar productos.
    • Registrar ventas.
    • Mostrar la lista de productos con sus precios.
    • Ejecutar la predicción de precios y mostrar el resultado de la regresión lineal.

Opción 2: Presentación con Interacción en Consola (Base 3.5)

  • Si los estudiantes optan por no desarrollar una interfaz gráfica, el sistema puede ser presentado mediante interacción a través de la consola.
  • Todas las funcionalidades del sistema (gestión de productos, ventas, y predicción de precios) deben estar presentes y ser accesibles desde la consola.
